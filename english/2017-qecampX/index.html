What is home automation?

Triggered by an event

Optional conditions based on the current status of the devices

Action which can control a device or external service

  • Arrive or leave the house
  • Voice command
  • Presence detection
  • Motion sensor
  • Weather conditions
  • Time/Sunset/Sunrise
  • On week days
  • When nobody is at home
  • After X minutes
  • If a test build failed
  • Heading to home
  • Arrived at work
  • Arm alarm
  • Find my phone
  • Turn on my car
  • Slow down fan
  • Increase temperature
  • Event Bus: facilitates the firing and listening of events – the heart  of Home Assistant.
  • State Machine: keeps track of the states of things and fires a state_changed event when a state has been changed.
  • Service Registry: listens on the event bus for call_service events and allows other code to register services.
  • Timer: sends a time_changed event every 1 second on the event bus.

# fdt_ptz-control.py
def run_query(scope, verbose=True, show_url=False):
    url = "http://{0}/cgi-bin/hi3510/param.cgi?cmd={1}&-usr={2}&-pwd={3}".format(
            HOSTNAME, scope, USERNAME, PASSWORD)

    req = requests.get(url)
    if req.ok and req.status_code == 200:
        if verbose:
            print(scope)
            if show_url:
                print(url)
            print('-' * len(scope))
            print(req.text)

def goto_preset(position, verbose=True, show_url=False):
    if isinstance(position, int):
        if position > 0 and position <= 8:
            # preset position starts on 0
            cmd = "preset&-act=goto&-number={}".format(position - 1)
            run_query(cmd, verbose, show_url)


if __name__ == '__main__':
    parser = argparse.ArgumentParser(description='FDT PTZ Control camera wrapper')
    parser.add_argument('preset', type=int, help='PTZ Preset [1-8]')
    args = parser.parse_args()

    if args.preset:
        goto_preset(args.preset, verbose=False, show_url=False)
